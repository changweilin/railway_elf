# Train Icon Assets

Generated top-down raster icons for every `train.type` currently emitted by `public/assets/rail-data.js`.

- Final icons are 256 x 256 PNG files with transparent backgrounds.
- All icons point upward in local image space; callers should rotate the image by the train heading.
- `train-icon-map.json` maps exact `train.type` strings to icon paths and the existing marker kind family.
- `train-icons-contact-sheet.png` shows each asset at 256, 64, 32, and 24 px for quick legibility checks.
